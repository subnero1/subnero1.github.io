UnetStack

The Underwater Network Stack

What is UnetStack?

UnetStack is a powerful and flexible agent-based software framework designed for underwater communication networks. It enables rapid design, simulation, testing, and deployment of software-defined underwater networks. Whether running on a Subnero modem or on a host computer connected to the modem, UnetStack transforms the modem into a sophisticated network node. Additionally, it can act as a gateway to terrestrial, satellite, or Internet-connected networks.

Components of UnetStack

UnetStack is designed with a comprehensive suite of tools and components to support underwater network development.
Simulator
Simulate network operations and optimize performance before deployment.
Messaging Fabric
Robust messaging framework for agent interactions.
Network Stack
Provides layers of network communication for underwater devices.
Channel Models
Integration with realistic underwater acoustic channel models like Bellhop, Kraken etc.

Key Features

Flexible Agent-Based Architecture
UnetStack consists of modular software agents that deliver well-defined services, creating an ecosystem where agents interact seamlessly. This approach ensures a clean separation of concerns while enabling the efficient sharing of information and services between agents.
Versatility in Deployment
The framework supports both Subnero and third-party modems, making it a customizable solution for diverse underwater communication needs.
Simulation & Real-world Deployment
UnetStack allows network configurations to be rapidly simulated and tested, providing a smooth transition from development to real-world underwater deployments.

Simulation Capabilities

Simulation is crucial in underwater network development, offering insights into performance before costly sea trials. UnetStack offers various levels of simulation

Unet Sim

Ideal for network layer developments, providing protocol-level simulation for large underwater networks.

VirtualAcousticOcean

Integration with underwater channel models like Bellhop for physical layer simulations, ensuring realistic underwater acoustic propagation.

Hardware-in-the-Loop Simulation

Extend your networks using VirtualAcousticOcean for testing with actual hardware in the loop. UnetStack4 or above required.

This comprehensive simulation suite allows developers to refine algorithms in a virtual environment and seamlessly transition to deployment on UnetStack-compatible modems.
